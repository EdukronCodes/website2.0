## 🎯 Quick Start

### 1. Run Locally
```bash
# Python 3
python -m http.server 8000

# or with Python 2
python -m SimpleHTTPServer 8000
```

### 2. Open Browser
```
http://localhost:8000
```

### 3. Start Learning!
- Click a category
- Browse topics
- Use search
- Toggle dark mode

## 🛠️ Customization Options

### Add New Category
1. Edit `js/navigation.js`
2. Add category configuration
3. Update HTML navbar
4. Create `docs/category-name/` directory
5. Add markdown files

### Add New Topics
1. Create markdown files in category directory
2. Files auto-appear in sidebar!
3. Follow naming: `NNN-topic-name.md`

### Change Styling
- Edit CSS variables in `css/style.css`
- Modify `css/sidebar.css` for sidebar
- Adjust `css/responsive.css` for mobile

### Deploy
- GitHub Pages (free)
- Netlify (free)
- Vercel (free)
- Any static host
