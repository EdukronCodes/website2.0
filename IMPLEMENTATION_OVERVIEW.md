
## 🛠️ Customization

### Add New Category
```javascript
// Edit js/navigation.js
const categories = {
  'your-category': {
    name: 'Your Category',
    emoji: '🎯',
    topics: ['Topic 1', 'Topic 2', ...]
  }
}
```

### Add New Topics
```bash
# Create markdown file
docs/your-category/001-topic-name.md

# Topic appears automatically in sidebar!
```

### Change Theme
```css
/* Edit css/style.css */
:root {
  --color-primary: #your-color;
  --accent-color: #your-accent;
}
```
